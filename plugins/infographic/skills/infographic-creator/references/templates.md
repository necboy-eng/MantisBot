## 模板选择指南

### 根据内容类型选择

- **时间线/发展历程** → `sequence-timeline-*`, `list-row-horizontal-*`
- **步骤流程** → `sequence-steps-*`, `sequence-stairs-*`
- **交互流程/时序图** → `sequence-interaction-*`
- **列举要点** → `list-row-*`, `list-column-*`, `list-grid-*`
- **二元对比** → `compare-binary-*`
- **SWOT 分析** → `compare-swot`
- **四象限分析** → `compare-quadrant-*`
- **组织架构** → `hierarchy-structure`, `hierarchy-tree-*`
- **思维导图** → `hierarchy-mindmap-*`
- **关系流程** → `relation-dagre-flow-*`
- **数据统计** → `chart-*`
- **关键词展示** → `chart-wordcloud`

### 根据视觉风格选择

- **简洁风格** → 选择带 `simple` 的模板
- **卡片风格** → 选择带 `card` 的模板
- **徽章风格** → 选择带 `badge` 的模板
- **动画效果** → 选择带 `animated` 的模板
- **紧凑布局** → 选择带 `compact` 的模板
